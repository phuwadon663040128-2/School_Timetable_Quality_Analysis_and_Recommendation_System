# WIREFRAMES.md

## Purpose
Low-fidelity wireframes สำหรับ MVP ของระบบ โดยเน้น desktop-first, Thai-first และ workflow โรงเรียนไทย

├───────────────┬─────────────────────────────────────────────────────────────┤

├─────────────────────────────────────────────────────────────────────────────┤
│ ┌────┬──────────────┬────────────┬────────────┬───────────┬────────────┐  │
│ │ #  │ ตาราง         │ สถานะ      │ Conflict   │ Score     │ Action     │  │
│ ├────┼──────────────┼────────────┼────────────┼───────────┼────────────┤  │
│ │ 1  │ ตาราง ม.ต้น   │ submitted │ 0 hard     │ 82        │ Review     │  │

│ └────┴──────────────┴────────────┴────────────┴───────────┴────────────┘  │

│ [Approve] [Return] [Publish]                                                 │
│ Return Reason: [.........................................................]   │

Rules:
- Return ต้องมี reason
- Publish ใช้ได้เฉพาะ approved และไม่มี hard conflict
- Approved ยังไม่เท่ากับ published
