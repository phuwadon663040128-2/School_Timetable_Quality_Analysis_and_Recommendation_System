# TPSS Lessons Learned

- Master Data ลดการกรอกซ้ำ
- RBAC และหลายบทบาทช่วยให้ workflow ชัด
- Conflict Check ต้องเป็นแกนหลักก่อน generator
- Smart Warning ช่วยให้ระบบไม่ rigid เกินไป
- Dashboard/Alerts ทำให้ผู้ใช้เห็น readiness ของข้อมูล
- Audit Trail สำคัญเมื่อหลายคนแก้ตาราง
- Seeder/Test Scenario ช่วย demo และ test ได้ดี
