# GLOSSARY.md

| Term | Meaning |
|---|---|
| Master Data | ข้อมูลกลางที่ใช้ร่วมกันทั้งระบบ |
| Class Section | ห้องเรียน เช่น ม.1/1 |
| Room | ห้องหรือสถานที่จริง |
| Room Type | ประเภทห้อง เช่น ห้องคอม ห้องวิทย์ |
| Shared Room | สถานที่ใช้ร่วมกันได้ |
| Timeslot | วันและคาบเรียน |
| Teaching Assignment | การกำหนดว่าครูสอนวิชาอะไรให้ห้องไหน |
| Timetable Entry | รายการตารางที่ถูกจัดลงคาบจริง |
| Hard Constraint | ข้อห้ามที่ผิดไม่ได้ |
| Soft Constraint | เงื่อนไขเชิงคุณภาพ |
| Quality Score | คะแนนคุณภาพตาราง |
| Recommendation | ข้อเสนอปรับปรุง |
| Homeroom Teacher | ครูประจำชั้น |
| Academic Staff | ฝ่ายวิชาการ/ผู้จัดตาราง |
