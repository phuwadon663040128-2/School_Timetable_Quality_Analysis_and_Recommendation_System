# ต้นแบบหน้าจอ Mock-up

Static HTML mock-up สำหรับสาธิตภาพรวมระบบและแม่แบบหน้าจอตามบทบาท โดยยังไม่เชื่อม Laravel หรือฐานข้อมูลจริง

## วิธีเปิด
เปิดไฟล์นี้ใน browser:

```txt
mockups/index.html
```

ถ้าต้องการ icon จาก Lucide ให้เชื่อมอินเทอร์เน็ตตอนเปิดหน้า เพราะ mock-up ใช้ CDN สำหรับ icon เท่านั้น ส่วน UI หลักยังใช้งานได้แม้ icon ไม่โหลด

## สิ่งที่ mock-up จำลอง
- ตัวสลับบทบาท
- เมนูด้านข้างตามบทบาท
- ภาพรวมตามบริบทของบทบาท
- ข้อมูลกลาง
- ภาระสอน
- หน้าจัดตารางสอนแบบปรับเอง
- ตรวจตารางชนและคุณภาพ
- คำแนะนำการปรับตาราง
- ตรวจสอบโดยกลุ่มสาระ
- รายการรออนุมัติ
- มุมมองตารางหลังเผยแพร่
- รายงาน / พิมพ์ PDF

## บทบาทที่รองรับ
- ผู้ดูแลระบบ
- ฝ่ายวิชาการ
- หัวหน้ากลุ่มสาระ
- หัวหน้างานวิชาการหรือรองฯ วิชาการ
- ครูผู้สอน
- ครูประจำชั้น
- ผู้บริหาร / ผู้ดูแลภาพรวม

## Demo flow ที่แนะนำ
1. เลือกบทบาท `ฝ่ายวิชาการ`
2. เปิด `จัดตารางสอน`
3. คลิกช่องตาราง
4. กด `ตรวจตารางชน`
5. กด `วิเคราะห์`
6. เปิด `คำแนะนำ`
7. กลับไป `ส่งอนุมัติ/เผยแพร่` แล้วกด `ส่งอนุมัติ`
8. เปลี่ยนบทบาทเป็น `หัวหน้างานวิชาการ / รองฯ วิชาการ`
9. กด `อนุมัติ`
10. กด `เผยแพร่`
11. เปลี่ยนบทบาทเป็น `ครูผู้สอน`, `ครูประจำชั้น` หรือ `ผู้บริหาร / ผู้ดูแลภาพรวม` เพื่อดูแม่แบบหน้าจอแต่ละบทบาท

## ขอบเขต
mock-up นี้เน้นภาพรวม UX และ ตรรกะสิทธิ์การใช้งาน เท่านั้น ไม่มี backend, validation จริง, auth จริง หรือ database
